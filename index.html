    # ExportRim Documentation

    ExportRim provides exporters a complete resource-integrated platform to develop, manage and expand international sales. ExportRim solution is available online on a website, mobile apps, and an upcoming desktop app. This user guide is designed for the website platform which has the most comprehensive features.

    ExportRim platform comprises of the following primary components:

    • Information resources
