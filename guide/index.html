            # Introduction

            # Welcome to ExportRim

            ExportRim is designed to provide a simplified standard user interface to enable you to focus on building your business. Each component adopts two main layouts, namely a card summary and a form dialog.

            Once you have set up an account, you can start to use all the free services for the basic membership. Features for paid subscribers are available automatically depending on the membership level.

            All your available components will be displayed on the sidebar if you are using the desktop website interface. If you access the website with a smaller display, for example with a phone or tablet, you will need to access the side menu with the dropdown icon.
